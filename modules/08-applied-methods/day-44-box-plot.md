# Day 44: Box Plot

*Five numbers. One visual. Everything you need to compare distributions.*

---

## Learning Objective

Understand how box plots summarize numerical distributions and why they are essential for comparing groups.

---

## The Concept

A **box plot** (box-and-whisker plot) displays five key statistics:

1. **Minimum** (excluding outliers)
2. **Q1** — 25th percentile
3. **Median** — 50th percentile (the line inside the box)
4. **Q3** — 75th percentile
5. **Maximum** (excluding outliers)

Points beyond the whiskers are **outliers**.

### Why box plots matter:

- **Compare groups side-by-side** — salaries across departments
- **Spot spread differences** — some groups are tight, others scattered
- **Find outliers instantly** — dots beyond the whiskers
- **See skewness** — median line closer to one end of the box

### Box plot vs histogram:

- Histogram → detailed distribution shape
- Box plot → compact summary, great for group comparisons

---

## Real-World Example

Comparing delivery times across cities:

- Mumbai: median 3 days, tight box
- Delhi: median 4 days, wide box with outliers up to 12 days
- Bangalore: median 3.5 days, moderate spread

Decision: Delhi's delivery network needs investigation — high variability and outliers suggest reliability issues.

---

## 🔑 Key Takeaway

> A box plot shows center, spread, and outliers in one compact visual. It is the best chart for comparing distributions across groups.
