## The Big Idea

Most real-world analysis time is spent understanding and cleaning data — not building models. This module covers the essential skills that separate good analysts from beginners:

- **Understanding data structure** — rows, columns, types
- **Assessing data quality** — missing values, outliers, inconsistencies
- **Preparing data for analysis** — cleaning, transforming, exploring

> Every analysis starts with one question: *Can I trust this data?*
