# Case Study: Do Discounts Drive Retention?

## Business Question

Do higher discount rates lead to more repeat purchases? Should we increase discount depth to improve customer retention?

## Status: Complete

## Planned Approach

1. **Data:** Synthetic e-commerce dataset — 2,000 customers, discount tier, purchase count, return flag
2. **Analysis:** Chi-square test, grouped retention rates, visualization by discount tier
3. **Expected Insight:** No significant relationship; highest discount tier may have LOWEST return rate
4. **Business Recommendation:** Invest in customer experience, not deeper discounts

## Key Output

- 1 grouped bar chart (retention rate by discount tier)
- 1 chi-square test result
- Shareable finding: *"Customers receiving 30%+ discounts had the lowest return rate (22%) — discounts attract deal-seekers, not loyal customers"*
