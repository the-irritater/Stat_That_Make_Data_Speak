# Case Study: Screen Time vs Productivity

## Business Question

Does higher daily screen time correlate with lower productivity scores? Should companies implement screen time policies?

## Status: Complete

## Planned Approach

1. **Data:** Synthetic dataset — 500 employees, daily screen time (hours) vs productivity score (1–100)
2. **Analysis:** Scatter plot, Pearson correlation, segmented analysis (low/medium/high screen time)
3. **Expected Insight:** Negative correlation above 6-hour threshold, minimal impact below
4. **Business Recommendation:** Focus on screen time quality (task type), not just quantity

## Key Output

- 1 scatter plot with correlation line
- 1 segmented bar chart (productivity by screen time tier)
- Shareable finding: *"Employees with >6 hours of daily screen time scored 23% lower on productivity metrics"*
