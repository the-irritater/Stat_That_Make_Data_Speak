## Visual Identity (for Carousels & Graphics)

### Color Palette

| Usage | Color | Hex |
|-------|-------|-----|
| Primary (headers, accents) | Deep Blue | `#1A365D` |
| Secondary (highlights) | Teal | `#2B6CB0` |
| Accent (CTAs, emphasis) | Coral | `#E53E3E` |
| Background (dark mode) | Charcoal | `#1A202C` |
| Background (light mode) | Off-White | `#F7FAFC` |
| Text (dark mode) | Light Gray | `#E2E8F0` |
| Text (light mode) | Dark Gray | `#2D3748` |

### Typography

| Usage | Font | Where to Get |
|-------|------|-------------|
| Headlines | **Inter Bold** | Google Fonts |
| Body text | **Inter Regular** | Google Fonts |
| Code | **JetBrains Mono** | Google Fonts |

### Carousel Branding

Every carousel should have:
- Your name on Slide 1 (bottom)
- "Stats That Make Data Speak" on the final slide
- Consistent color scheme (use the palette above)
- Slide numbers (e.g., "3/7")

## Hashtag Strategy

### Primary (use on every post)
```
#Statistics #DataAnalytics #Python
```

### Secondary (rotate based on content type)
```
#DataScience #StatsInAction #LearningInPublic #DataAnalyst
```

### Specific (use when relevant)
```
#Correlation #Regression #HypothesisTesting #EDA #Visualization
```

### Rule: 3–5 hashtags per post. Never more than 7.

## Content Voice

| Do | Don't |
|----|-------|
| Simple, direct language | Academic jargon |
| Business examples | Textbook exercises |
| "Here's what I found" | "In conclusion, the data suggests" |
| Specific numbers and findings | Vague generalizations |
| Conversational tone | Formal paper tone |
| Acknowledge limitations | Overstate confidence |
