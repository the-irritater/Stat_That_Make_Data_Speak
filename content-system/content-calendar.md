# Content Calendar

## Weekly Rotation (2 posts/week minimum, 3rd optional)

| Day | Type | Pillar | Visibility Loop |
|-----|------|--------|----------------|
| **Monday** |  Concept / Carousel | Theory simplified | Link to module in repo |
| **Thursday** | Applied Post | Python + Business Data | Link to notebook + show chart |
| **Saturday** *(optional)* | Case Study / Insight | Proof of Work | Link to case study + key finding |

> Consistency > Intensity. 2 solid posts beat 5 rushed ones.

## Batching Strategy

Spend **60–90 minutes once per week** creating all posts:

1. **Sunday evening:** Write 2–3 post drafts using templates
2. **Schedule:** Use LinkedIn's native scheduler or Buffer
3. **Daily (10 min):** Engage — comment on 10 data/analytics posts
4. **Monthly:** Review analytics, double down on what worked
