# Day 28: Regression

## LinkedIn Post Copy
```text
Day 28 of Stats That Make Data Speak

Topic: Regression

Regression is used to understand and predict relationships between variables.

Simple linear regression studies the relationship between one independent variable and one dependent variable.

Real-life example:

A company wants to predict sales based on advertising budget.

Advertising budget = independent variable  
Sales = dependent variable  

Regression helps answer:

If advertising budget increases, how much will sales increase?

Another example:

Predicting house price using area.

Area = independent variable  
House price = dependent variable  

Key takeaway:

Regression helps us understand how one variable affects another and predict future values.
```
