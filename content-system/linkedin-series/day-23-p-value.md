# Day 23: p-value

## LinkedIn Post Copy
```text
Day 23 of Stats That Make Data Speak

Topic: p-value

p-value is one of the most misunderstood concepts in statistics.

In simple words:

The p-value tells us how surprising the observed result is if the null hypothesis is true.

A small p-value means the result is unlikely under the null hypothesis.

Common rule:

If p-value < 0.05, the result is considered statistically significant.

Real-life example:

A company tests a new advertisement.

The new ad gets more clicks than the old ad.

But is the difference real or just random?

The p-value helps answer this.

Key takeaway:

p-value does not prove truth.

It helps measure evidence against the null hypothesis.
```
