# Day 25: t-test

## LinkedIn Post Copy
```text
Day 25 of Stats That Make Data Speak

Topic: t-test

A t-test is used to compare means.

It helps us check whether the difference between two averages is statistically meaningful.

Real-life example:

A teacher wants to know whether a new teaching method improves student marks.

Group A learns using the old method.

Group B learns using the new method.

After the exam, we compare the average marks of both groups.

If Group B has a higher average, is it truly because of the new method?

Or could it be due to random variation?

A t-test helps answer this.

Common types:

One-sample t-test  
Independent samples t-test  
Paired samples t-test  

Key takeaway:

A t-test helps compare averages scientifically.
```
