# Day 4: Types of Data

## LinkedIn Post Copy
```text
Day 4 of Stats That Make Data Speak

Topic: Types of Data

Before analyzing data, we must understand its type.

Data is mainly divided into two types:

Qualitative data and quantitative data.

Qualitative data describes categories.

Examples:

Gender, city, education level, product category, feedback type.

Quantitative data represents numbers.

Examples:

Age, income, height, marks, sales, screen time.

Real-life example:

In a student survey:

Department = qualitative  
Age = quantitative  
Daily screen time = quantitative  
Gender = qualitative  
Exam marks = quantitative  

Why does this matter?

Because data type decides which statistical method we should use.

We can calculate average age.

But we cannot calculate average gender.

Key takeaway:

Before applying any statistical method, first understand the type of data.
```
