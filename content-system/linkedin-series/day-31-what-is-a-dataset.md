# Day 31: What is a Dataset?

## LinkedIn Post Copy
```text
Day 31 of Stats That Make Data Speak

Topic: What is a Dataset?

A dataset is a collection of data arranged in a structured form.

Most commonly, it looks like a table.

Rows represent observations.
Columns represent variables.

Real-life example:

In a student survey, each student is one row.

Age, gender, department, marks, and screen time are columns.

So, if 500 students answer 10 questions, the dataset has 500 rows and 10 columns.

Data analytics use:

Before creating charts, dashboards, statistical tests, or machine learning models, we first need to understand the dataset structure.

Key takeaway:

A dataset is not just a table.

It is the starting point of every data-driven decision.
```
